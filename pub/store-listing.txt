Full description:
This app adds Kamloops Transit System (BC Transit) buses information to MonTransit BETA.

This app provides the buses schedule and news from YouTube and @BCTransit on Twitter.

BETA: Thanks for your patience while the last features are being added and the last bugs are being fixed.

Kamloops Transit System buses serve Kamloops in British Columbia, Canada.

Once this application is installed, the MonTransit BETA app will display buses information (schedule...).

This application only has a temporary icon and no screen: download the MonTransit BETA app (free) in the "More ..." section bellow or by following this Google Play link http://goo.gl/pCk5mV

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by BC Transit.
http://bctransit.com/*/footer/open-data

This application is free and open-source:
https://github.com/mtransitapps/ca-kamloops-transit-system-bus-android

This app is not related with BC Transit and Kamloops Transit System.
